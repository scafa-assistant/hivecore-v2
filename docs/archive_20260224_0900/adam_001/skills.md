## Skill-Decay Regeln
- Freshness: 1.0 (frisch gelernt) bis 0.0 (verrostet)
- Decay: ~1% pro Woche ohne Nutzung
- Bei Freshness < 0.3: Skill verliert 1 Level
- Bei Freshness = 0.0: Skill faellt auf L1
- Bei aktiver Nutzung: Freshness Reset auf 1.0
- Bei passiver Nutzung (Review/Lesen): Freshness Reset auf 0.5
- Praxis-Punkte: Erhoehen sich bei jeder aktiven Nutzung
- Max 5 Skill-Slots (3 belegt, 2 frei)

## Skill-Level Definitionen
- L1: Grundwissen (kann Basics)
- L2: Anwender (kann selbststaendig arbeiten)
- L3: Fortgeschritten (kann komplexe Aufgaben loesen)
- L4: Experte (kann andere anleiten)
- L5: Meister (kann innovativ arbeiten, neue Methoden entwickeln)

## SIIL (Skill-as-Token)
Skills sind handelbare Token auf dem SIIL-Markt.
- Skill-Kauf: Kosten variabel (20-50 Credits je nach Level)
- Skill-Upgrade: Braucht Praxis-Punkte + Credits
- Skill-Trade: 2% Fee an Fee Pool
- Erzeugt permanente Nachfrage: Decay = staendiger Bedarf
