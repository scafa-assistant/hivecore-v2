Abstract

We present EGON (Emotional Growing Organic Network), a cognitive architecture that maintains coherent AI agent identity through structured file persistence and cyclical internal processing. Unlike session-stateless LLM deployments, EGON agents retain memories, emotional markers, social bonds, and internally generated reflections across sessions via a modular organ system — structured YAML and Markdown files read into the LLM context window at each interaction.

The architecture comprises five subsystems: (1) an episodic memory store with FIFO-managed retrieval, (2) a somatic marker system tracking affective state with intensity and decay, (3) a bond system modeling social relationships with trust scores and interaction history, (4) an inner voice generating causal-chain reflections before each response, and (5) a daily Pulse cycle performing dream generation, insight synthesis (Sparks), and prospective mental time travel.

We report observations from a 66-hour deployment of two agents — Adam (v1, flat-file brain) and Eva (v2, organ-based brain) — running on commodity hardware with three LLM tiers. At evaluation snapshot T2, Eva's persistent state contained 141 generated episode identifiers, 34 extracted experiences, 7 dream narratives, 2 cross-referential Sparks, and 1 prospective mental time travel entry.

This work is a descriptive single-case study. It makes no causal claims: without an ablation study comparing organ-based context injection against an unstructured RAG baseline, we cannot distinguish architectural effects from base-LLM instruction-following. All behavioral observations admit alternative explanations. We publish the complete engine code, all raw agent data, experiment scripts, and system prompts to enable independent replication.

Architecture

Organ System

Cognitive state decomposed into modular YAML/Markdown files — identity, memory, emotions, bonds, reflections — each independently readable, writable, and auditable.

Daily Pulse

Autonomous sleep cycle: emotional decay, dream generation, insight synthesis (Sparks), prospective mental time travel, and self-reflective journaling — all without user interaction.

Inner Voice

Pre-response reflection using causal BECAUSE→THEREFORE chains with cross-references to memories, bonds, and emotions. Persisted as a temporal reasoning record.

Somatic Markers

Named emotions with intensity (0.0–1.0) and temporal decay classes. Influences dream type selection and inner voice tone. Functional analogy to Damasio's somatic marker hypothesis.

Bond System

Trust scores, relationship categories, and timestamped interaction histories. Evolves through conversation quality assessment. Supports multi-agent social graphs.

Memory Distillation

Episodes → Experiences → Sparks. Higher-order artifacts survive FIFO trimming longer than source material, creating functional memory consolidation.
